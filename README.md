# UART Receiver using Verilog HDL

## Overview
This repository contains the Verilog HDL implementation of a UART (Universal Asynchronous Receiver Transmitter) Receiver along with its testbench for functional verification.

## Modules Included
- UART Receiver
- Testbench

## Features
- Serial data reception
- Configurable baud rate
- Functional verification using testbench
- Simulation using Verilator
- Waveform analysis using GTKWave

## Tools Used
- Verilog HDL
- Verilator
- GTKWave
- Git & GitHub

## Files
- uart_receiver.v
- uart_receiver_tb.v
- run.sh

## Applications
- Embedded Systems
- FPGA Design
- Serial Communication
